JavaScript 预编译

JavaScript 从预编译到执行的过程，大致可以分为四步：

  1. 词法分析
  2. 语法分析：检查代码是否存在错误，若有错误，引擎会抛出语法错误。同时会构建一颗抽象语法树(AST)
  3. 预编译
  4. 解释执行

预编译

JavaScript 是解释性语言，也就是说，编译一行，执行一行，但是 JS 并非上来就进入编译环节，它在编译之前存在预编译过程。

JS 中预编译一般有两种：全局的预编译和函数的预编译，分别发生在 script 内代码执行前和函数执行前。
